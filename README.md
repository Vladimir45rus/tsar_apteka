# 💊 Царь Аптечка

Бот-ассистент для контроля приёма лекарств и быстрого вызова экстренных служб. Входит в экосистему «Царь».

## 🚀 Функции

- Напоминание о приёме лекарств (push-уведомления)
- Ведение дневника самочувствия
- Быстрый вызов скорой помощи (с передачей геолокации)
- Проверка совместимости препаратов (база знаний)
- Хранение истории назначений

## 🛠 Стек технологий

- **Бэкенд:** Python + FastAPI
- **Фронтенд:** React (веб-версия), Flutter (мобильное приложение — в планах)
- **База данных:** SQLite (прототип) → PostgreSQL
- **Уведомления:** Firebase Cloud Messaging
