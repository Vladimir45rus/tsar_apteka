# 💊 Царь Аптечка (Tsar-Pharmacy)

Бот-ассистент для контроля приёма лекарств и быстрого вызова экстренных служб. Входит в экосистему «Царь».

## 🚀 Функции
- **Контроль:** Напоминание о приёме лекарств (push) и ведение дневника самочувствия.
- **Безопасность:** Быстрый вызов скорой помощи с передачей геолокации.
- **Интеллект:** Проверка совместимости препаратов и хранение истории назначений.
